# CopilotHydra branding assets

These assets were generated from the user-provided `logo.png` source in the repository root.

## Files

- `copilothydra-logo-original.jpg` — preserved original upload bytes (the supplied `logo.png` is actually a JPEG-encoded square image)
- `copilothydra-logo-1024.png` — canonical lossless repo logo for docs and GitHub usage
- `copilothydra-logo-512.png` — practical square size for README embeds or GitHub repository social preview prep
- `copilothydra-app-icon-256.png` — smaller square app/icon asset
- `copilothydra-app-icon-128.png` — compact icon asset
- `copilothydra-icon-transparent-1024.png` — best-effort transparent icon-only export from the flattened source
- `copilothydra-icon-transparent-512.png` — transparent icon variant for general app/repo usage
- `copilothydra-icon-transparent-192.png` — practical PWA/app icon size
- `copilothydra-icon-transparent-180.png` — Apple-touch style size
- `copilothydra-favicon-64.png` — large favicon/icon size
- `copilothydra-favicon-48.png` — medium favicon/icon size
- `copilothydra-favicon-32.png` — common browser favicon size
- `copilothydra-favicon-16.png` — smallest browser favicon size

## GitHub usage notes

- Repository avatar / social preview changes still need to be applied manually in the GitHub web UI.
- Recommended asset for a repo/org avatar: `copilothydra-logo-512.png`
- Recommended asset to start from for other downstream exports: `copilothydra-logo-1024.png`
- Recommended transparent/icon-only asset: `copilothydra-icon-transparent-512.png`
- Recommended favicon sizes: `copilothydra-favicon-32.png` and `copilothydra-favicon-16.png`

## Quality note

- Transparent/icon-only extraction is **best-effort** because the supplied source is an opaque flattened image. Background removal and icon isolation were generated heuristically, so some soft edges/glow may remain.
